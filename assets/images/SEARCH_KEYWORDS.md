# Transyes Image Search Keywords

## How To Use

- Photo-style images:
  - First try `Pixabay`
  - Then try `Pexels`, `Unsplash`, or other free commercial-use libraries if needed
- Icon-style assets:
  - First try `Iconfont`, `Flaticon`, `SVG Repo`, or other free icon libraries
- Suggested search method:
  - Search English keywords first for stock photos
  - Search Chinese keywords first for domestic icon libraries
- File format suggestion:
  - Photos: prefer `jpg`
  - Transparent icons: prefer `png` or `svg`

## Notes

- Do not use AI-generated fake client logos
- Do not replace real QR codes with stock images
- For each file below, you can search one keyword set first, then try alternates if the result quality is poor
